📘 Assignment B1 – Facebook Dataset (Pandas Operations)
Commands & Explanations
Python
import pandas as pd
➤ Imports the pandas library and gives it the alias 'pd'. Pandas is used for data manipulation and analysis using DataFrames.

Python
df = pd.read_csv('dataset_Facebook.csv', delimiter=';')
➤ Reads the Facebook CSV file into a DataFrame 'df'. The delimiter=';' tells pandas that columns are separated by semicolons instead of commas.

Python
df
➤ Displays the entire DataFrame — shows all rows and columns of the loaded dataset (500 rows × 19 columns).

Python
df.describe()
➤ Generates descriptive statistics for all numeric columns: count, mean, standard deviation, min, 25th/50th/75th percentile, and max values.

Python
df_subset_1 = df[['like','share']]
➤ Creates a subset DataFrame containing only the 'like' and 'share' columns from the original DataFrame.

Python
df_subset_1
➤ Displays the subset DataFrame with 500 rows and 2 columns (like and share).

Python
df_subset_2 = df[['comment','Type']]
➤ Creates another subset with only 'comment' and 'Type' columns from the original DataFrame.

Python
df_subset_2
➤ Displays the second subset DataFrame with 500 rows and 2 columns (comment and Type).

Python
merged_data = pd.merge(df_subset_2, df_subset_1, left_on='comment', right_on='like')
➤ Merges the two subsets by matching values of 'comment' column from df_subset_2 with values of 'like' column from df_subset_1. This is an inner join by default — only rows where comment equals like are kept. Result has 1467 rows and 4 columns.

Python
merged_data
➤ Displays the merged DataFrame showing comment, Type, like, and share columns.

Python
merged_data.sort_values(by=['like'], ascending=False)
➤ Sorts the merged DataFrame in descending order based on the 'like' column. Rows with highest likes appear first.

# Method 1

Python
merged_data.transpose()
➤ Transposes the merged DataFrame — rows become columns and columns become rows. Result: 4 rows × 1467 columns.

# Method 2

Python
merged_data.T
➤ Another way to transpose using the .T attribute — functionally identical to .transpose(). Both give 4 rows × 1467 columns.

Python
df
➤ Displays the original full DataFrame again (500 rows × 19 columns) for reference.

Python
df.Type.unique()
➤ Returns an array of all unique values in the 'Type' column. Result: ['Photo', 'Status', 'Link', 'Video'] — shows the distinct post types present.

Python
pd.melt(df, id_vars=['Type'], value_vars=['comment'])
➤ Reshapes (unpivots) the DataFrame from wide format to long format. 'Type' is kept as-is (identifier variable) while 'comment' becomes a value variable. Result: 500 rows × 3 columns with columns Type, variable, value.
